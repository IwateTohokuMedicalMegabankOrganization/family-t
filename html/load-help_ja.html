family-tに家族歴を読み込むには、2つ方法があります。

  1. このサイトで作成した家族歴（MFHP）ファイルを読み込む
  2. Microsoft HealthVaultから健康情報を読み込む

家族歴ファイルの読み込み

family-tで保存した家族歴ファイルを読み込むことができます。読み込んだのち、より多くの情報を追加できます。 ファイルの名前は、FamilyHistory.xmlまたはname_FamilyHistory.xml（nameはファイルを保存した人の名前）です。 コンピュータ、ポータブルドライブ、またはストレージサービス（GoogleまたはDropbox）からファイルを読み込むことができます。

注：GoogleやDropboxなどの第三者システムに機密情報を保存する場合は、第三者システムの利用規約とプライバシーポリシーを確認して、安全であることを確認してください。

次の表は、家族歴ファイルを読み込む方法を示しています。 family-tで作成したファイルは、拡張子XMLのファイルのロードしかできません。

読み込み元 XMLファイルを読み込むには、下記手順に従ってください
コンピュータ ファイルから家族歴を読み込む
  1. ブラウズをクリック
  2. ファイルアップロード画面が表示されます。
  3. XMLファイルを探して選択します。
  4. ブラウズをクリック
  5. XMLファイルが読み込まれ、家族歴を更新することができます。
USBドライブなどのポータブルストレージデバイス ファイルから家族歴を読み込む
  1. ポータブルストレージデバイスが正しく接続されていることを確認してください。
  2. ブラウズをクリック
  3. ファイルアップロード画面が表示されます。
  4. ファイル選択ダイアログでデバイスを探します。
  5. XMLファイルを探して選択します。
  6. 開く/OKをクリック
  7. XMLファイルが読み込まれ、家族歴を更新することができます。
Googleドライブ Googleドライブから家族歴を読み込む
  1. Googleドライブから家族歴を読み込むをクリック
  2. Googleアカウントにログインします。
  3. XMLファイルが読み込まれます。
Dropbox Dropboxから家族歴を読み込む
  1. Dropboxから家族歴を読み込むをクリック
  2. Dropboxアカウントにログインします。
  3. 指示に従って、ファイルをロードします。
  4. XMLファイルが読み込まれ、家族歴を更新することができます。

Microsoft HealthVaultから家族歴を読み込む

Microsoft HealthVaultでは、オンラインで健康情報を収集、保存、および共有することができます。 HealthVaultアカウントの家族歴情報をfamily-tにコピーすることができます。

注：HealthVaultアカウントをお持ちでない場合は、このサイトで家族歴を保存する際にアカウントを作成するか、MicrosoftのHealthVaultサインアップ画面から作成することができます。

Microsoft HealthVaultから健康情報を読み込む

  1. Microsoft HealthVaultから健康情報を読み込むをクリック
  2. family-tはHealthVaultに接続します。
  3. HelthVaultカウントにログインします。
  4. 指示に従ってファイルをコピーします。
  5. XMLファイルが読み込まれ、family-tを更新することができます。
